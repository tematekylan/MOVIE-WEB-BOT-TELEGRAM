# App Tele Bot

Ce dossier contient le bot Telegram de l’application App Tele.

## Installation

1. Copier `.env` 
2. Ajouter votre token de bot Telegram dans `.env`
3. Lancer l’installation :

```bash
cd "t:\Programmation\Web App\App Tele\bot"
npm install
```

## Démarrage

```bash
npm start
```

## Commandes disponibles

- `/start` : accueil
- `/help` : aide
- `/films` : liste des films
- `/film <id>` : détails d’un film
- `/contact` : informations de contact
